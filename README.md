# Tresorticker

A barebones Node.js app using [Express](http://expressjs.com/) to parse the to check if a set of crypto trading parameters would lead to a buy or no buy decision of the trading engine. 

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) and the [Heroku Toolbelt](https://toolbelt.heroku.com/) installed.

```sh
git clone https://github.com/friedhelmensch/tresorticker.git # or clone your own fork
cd node-js-sample
npm install
npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
heroku create
git push heroku master
heroku open
```

Alternatively, you can deploy your own copy of the app using the web-based flow:
